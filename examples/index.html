Examples

Here are a few examples of using the jquery plugin. Of course, you have to include at least JQuery 1.4 and Google maps API V3.

<script src="http://code.jquery.com/jquery-latest.js" type="text/javascript"></script>
<script src="http://maps.google.com/maps/api/js?sensor=true" type="text/javascript"></script>

  • Simple map initialization (defaults to road map)

    Html Code:

    <div id="map" class="google-map"></div>

    Jquery Code:

    //Example 1: Simple declaration with default coordinates and zoom
    var map = $("#map").gmap3( {
      lat: 25.928407,
      lng: -79.727783,
      zoom: 7
    });
